Sitio web informativo y funcional para "WebCamp", una conferencia (o bootcamp) sobre desarrollo web. El proyecto implementa un diseño *responsive* y moderno, enfocado en proveer información a los asistentes, mostrar a los ponentes y gestionar el registro.

## 📋 Características Principales

* **Página de Inicio Atractiva:** Un *hero section* que captura la atención e información clave del evento.
* **Sección de Ponentes (Speakers):** Presentación de los expertos que participarán en el campamento.
* **Calendario / Agenda:** Un cronograma detallado de las charlas y talleres, organizados por día y categoría.
* **Sección de Precios:** Muestra los diferentes pases o boletos disponibles.
* **Formulario de Registro/Contacto:** (Asumiendo que existe) Un formulario funcional (posiblemente conectado a PHP y SQL) para que los usuarios se registren o envíen consultas.
* **Diseño Totalmente Responsivo:** Se adapta perfectamente a dispositivos móviles, tablets y computadoras de escritorio.

## 🛠️ Stack Tecnológico

Este proyecto fue construido principalmente con tecnologías frontend, con un posible componente de backend para la gestión de formularios.

* **Maquetación:** **HTML5** (utilizando semántica moderna).
* **Estilos:** **CSS3** (posiblemente con SASS o CSS Grid/Flexbox).
* **Interactividad:** **JavaScript (Vanilla)** para animaciones, sliders, o validación de formularios.
* **(Opcional) Backend:** **PHP** para procesar el formulario de registro/contacto.
* **(Opcional) Base de Datos:** **MySQL / PostgreSQL** para almacenar los datos de los usuarios registrados.

## 🔧 Instalación y Puesta en Marcha

Dado que este es principalmente un proyecto de maquetación, puedes ejecutarlo fácilmente en tu entorno local.

1.  **Clonar el repositorio:**
    ```bash
    git clone [https://github.com/FranciscoGG09/Web-Camp.git](https://github.com/FranciscoGG09/Web-Camp.git)
    cd Web-Camp
    ```

2.  **Abrir el proyecto:**
    * Simplemente abre el archivo `index.html` en tu navegador.
    * O (recomendado) usa una extensión como **Live Server** en VS Code para verlo con recarga automática.

3.  **(Si aplica) Configuración del Backend:**
    * Si el proyecto incluye PHP, deberás colocar la carpeta del proyecto dentro del directorio `htdocs` (XAMPP) o `www` (WAMP).
    * Importa el archivo `.sql` (si existe) a tu gestor de base de datos.
    * Asegúrate de configurar las credenciales de la base de datos en el archivo de conexión de PHP (ej. `db_conexion.php`).
